# 用户自定义词组+单词

针对四种数据，支持用户自定义数组。

在默认系统的文本库加载完成之后，可以添加自定义词组。

ps: 只提供一个 add(key, value)，新的 key 会覆盖旧的 key。

# 并行处理分词

如果传入的内容比较长，分词之后，较长的默认并行处理。

使用者无感知。

ps: 多线程处理性能提升不明显，因为每一个词语的转换很快。
